### loading a map
```
./fdf ./maps/nice_maps/t2.fdf
```
![t2.png](https://github.com/0815-alex/fdf/blob/master/img/t2.png)

### creating a map
```
./fdf astein
```
![astein.png](https://github.com/0815-alex/fdf/blob/master/img/astein.png)
